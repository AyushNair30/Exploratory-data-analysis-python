# Data Analysis Portfolio Project

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) using Python. The goal was to analyze the dataset, clean the data, identify trends, and extract meaningful insights using Pandas and NumPy.

## Objectives

* Load and inspect the dataset
* Clean and preprocess the data
* Handle missing values and duplicates
* Perform data exploration and analysis
* Generate insights from the dataset
* Practice real-world data analysis workflows

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook
* VS Code

## Skills Demonstrated

* Data Cleaning
* Data Manipulation
* Exploratory Data Analysis (EDA)
* Data Filtering and Sorting
* Grouping and Aggregation
* Working with Real-World Datasets

## Project Workflow

1. Imported and inspected the dataset
2. Checked data types and missing values
3. Cleaned and prepared the data
4. Performed exploratory analysis
5. Answered business/data-related questions
6. Extracted key insights
